Help

Welcome!

This application is meant to provide you with required information about the points of interest in the nearby area. You can check the route on map and see your friends locations in order to get to them easier.

Take your time to scroll down in order to find further information about the related functionalities.

Authentication

In order to access the location functionalities, you need to log into an existing account or create a new one. In case you don't have an account and you don't want to log in via the related social accounts, you can create a new account, by clicking the SignUp button, which can be found in the bottom of the home page. The home page is the first page of the application. You need to provide at least your username, your email address and a valid password. Pay attention when choosing the password, for it not to be too common or too similar with your name.

Log in

As mentioned above, you can only be provided with the functionalities related to POIs and friends management if you are logged in. You can opt for logging in via your social accounts, such as Google or Twitter. You can also log into an already existing account, if having one. You can select the log in option from the home page. The log in via Google or Twitter buttons are found in the bottom of the home page, while the simple log in option is to be found in the right upper part of the home page. If you possess a beacon, you can log in via it. Once you are logged in, you are redirected to your account page.

My account

There you can generate a QR code, in order to log in easier from other personal devices. You can also select to log out from the application. Also, you can get to the location page.

Friends

In the right upper part of the location page, you can find a variety of functionalities related to friends management. You can choose to add a new friend, remove an already existing friend or answer to a friend request. If you enable the location, by clicking the button with the same name from the left upper side of the page, you will be able to see the location of your friends on the map and your location as well. The location of each friend is represented on map with a pin, on which it is written his or her name.

Plugins

In the right upper part of the location page you can find the functionalities related to plugins. If you want a certain functionality to be fulfilled, you can opt for adding a new plugin. Once the pluggin is loaded, the name will appear in the list of plugins available in the bottom right side of the page. Once the plugin is not relevant for you anymore, it can be removed.

See nearby places

In the right bottom of the page you can see the available list of already existing plugins. By defult, you have at your disposal 5 plugins: See nearby places See nearby places types Search location by name Search taxis See friends locations You can opt for checking the nearby points of interest. In this case, select the plugin with the same name from the list. You need to select the radius which determines the distance on which you want to see the points of interest. It has to be greater than 5 meters and lower than 2000 meters. You also need to select the language in which the information is shown, consisting in English or Romanian. The last parameter for your search consists in the type of point of interest which you are interested in. You can filter your results by proeminence by selecting the "Rank by proeminence filter" and you can also show only the opened points of interest, by selecting the "Open now" option. The response consists in a list of points of interest which meet the requirements. Each entry contains reviews, the description and contact information.

See nearby places types

The second plugin can show you all the available types of points of interest on a specific area. The parameters needed for your search consist in the radius and the language. As in the previous example, the radius has to be greater than 5 and lower than 2000. Also, you can opt for showing only the types which correspond to opened points of interest. You will be provided with a list of types which meet the requirements.

Search location by name

If you want to check a specific location, you can do it by selecting the plugin with the same name. You need to input the name in the first field and the type of POI you are looking for. Also, you need to select the radius, in order to define an area for search. The radius needs the have a value in the interval (5, 2000) meters. Finally, you need to provide the language in which you want the information to be shown. You will get one or more results, which correspond to your search. These results will contain the address, the available time and reviews, when existing.

Search taxis

If you want to see all the available taxi companies, in order to move from one point to another, you can use the plugin with the same name. As in the previous examples, you will need to provide a radius with values between 5 and 2000 meters, as well as the language in which you want the information to be shown. You will get a list of results which meet the requirements, each entry containing reviews, description and contact information.

Search friends

If you want to check the location of your friends, you can do this using the plugin with the same name. You will need to input the distance on which you want to see friends and the value of time used for filtering by users activity. You will therefore see pins on maps, each of them pointing to the location of one friend.
